# 概要
スマホの利用を制限できるアラームアプリです。
<p align="center">
  <kbd><img src="https://github.com/s20019/ApplicationController/blob/master/smasto.gif" width="300"></kbd>
</p>

# 動作環境
- Androidスマホ
- Android 12.0 以上

# 使い方
15分区切り or 1時間区切りのどちらかのリストから、アラームが来る時間を選択。
